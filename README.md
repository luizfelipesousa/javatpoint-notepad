# Notepad - JavaTpoint

## Sobre

Projeto desktop de uma aplicação bloco de notas.

O foco principal de estudo é entender e aplicar os conceitos e fundamentos utilizados na biblioteca Java Swing.

Os principais pontos e tecnologias abordados neste projeto, foram:

- Montagem de componentes

- Controle de Eventos

- Manipulação de arquivos
